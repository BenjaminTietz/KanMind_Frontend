Privacy Policy

1. General Information

This Privacy Policy explains how personal data is processed when using the KanMind application. KanMind is a project developed as part of an educational program.

2. Educational Context

The frontend of this application is provided by the Developer Akademie as part of an educational training program. The frontend itself does not independently process or store personal data.

The backend and API services are developed and operated by the project participant for learning and demonstration purposes.

3. Data Controller

The data controller for personal data processed via the backend API is:

KanMind (Educational Project)
Project Owner (backend): Benjamin Tietz
Email: mail@benjamin-tietz.com

4. Personal Data Collected

The backend API processes only the data required to provide authentication and basic application functionality:

  • Email address
  • Full name (if provided)
  • Authentication tokens

5. Purpose of Data Processing

Personal data is processed exclusively for the following purposes:

  • User authentication
  • Access control to protected features
  • Security and system integrity

6. Authentication and Security

Authentication is handled via token-based authentication. Passwords are securely hashed and never stored in plain text. Tokens are used solely to maintain authenticated user sessions.

7. Cookies and Tracking

This application does not use cookies for tracking, analytics, or advertising purposes. No third-party tracking services are integrated.

8. Data Sharing and Third Parties

No personal data is shared with third parties. The application does not integrate external services that process user data.

9. Data Retention

Personal data is stored only for the duration necessary to support the educational use of the application. Users may request deletion of their data at any time.

10. User Rights

Users have the right to request:

  • Access to their personal data
  • Correction of inaccurate data
  • Deletion of their personal data

11. Changes to this Privacy Policy

This Privacy Policy may be updated to reflect changes in the educational project or legal requirements. The current version will always be available within the application.
